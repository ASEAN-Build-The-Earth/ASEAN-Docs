## ðŸ’¬ <small>What's the server version?</small>

For Singapore:

<details>
  <summary>Java version 1.18.2</summary>
  
  Singapore server is hosted on a vanilla version (Technically Purpur), so it can be updated to newer versions if possible.
</details>

For other countries:

<details>
  <summary>Java version 1.12.2</summary>
  
  Due to one of our mods (Cubic Chunks), which allows building above Minecraft's default height limit (256 blocks), our other servers are on 1.12.2.
</details>

Bedrock version:

<details>
  <summary>Latest</summary>
  
  Bedrock server automatically gets updated to the latest version. Note that Bedrock version only supports the Plot system server.
</details>

## ðŸ’¬ <small>I want to build this place</small>

<details>
  <summary>Become an official builder</summary>
  
  Due to our [Builder System](./getting-started/builder-system), new builders can only build in the [Plot system server](./getting-started/visiting#2-plot-server) with limited build sites to choose from. This prevents unfinished builds from cluttering the server and keeps the build team organized.
  
  You may also build a BuildTheEarth building by yourself in a singleplayer world with [BuildTheEarth Modpack](../guide/modpack) installed, but we **do not recommend** this method, as we cannot properly review your build.
</details>
